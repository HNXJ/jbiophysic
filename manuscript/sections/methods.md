## METHODS — OPTIMIZATION FRAMEWORK
We introduce a hybrid optimization framework combining Adaptive Gradient Synaptic Drift Regularization (AGSDR) with Genetic–Stochastic Gradient Descent (GSGD) to calibrate large-scale biophysical neural simulations. AGSDR is first applied to a single model instance to enforce physiological constraints, including target firing rates (5 ± 1 Hz), excitation–inhibition balance, and stable oscillatory dynamics in the gamma band (~40 Hz). This is achieved through gradient-based updates with clipped gradients and bounded parameter projections, ensuring that synaptic conductances remain within biologically plausible ranges. Following this calibration phase, the optimized parameter set is expanded into a population through stochastic perturbation, forming the initialization for GSGD. GSGD then performs iterative optimization by combining evolutionary operators (selection, crossover, mutation) with stochastic gradient descent updates computed via differentiable simulation. Gradients are estimated using truncated temporal windows of the simulation, introducing controlled stochasticity that improves scalability while preserving convergence properties. This hybrid approach leverages the global exploration capabilities of genetic algorithms and the local refinement efficiency of SGD, enabling convergence toward stable dynamical regimes rather than isolated parameter minima.

## METHODS — MULTI-AREA SIMULATION AND TRAINING
The cortical model consists of a hierarchical network of interconnected areas spanning primary sensory (V1) to prefrontal cortex (PFC), each implemented as a canonical cortical column with layer-specific populations and inhibitory interneuron subtypes (PV, SST, VIP). Synaptic interactions include conductance-based AMPA, NMDA, and GABAergic currents, with NMDA-mediated nonlinearities enabling coincidence detection and calcium-dependent plasticity. During the training phase, synaptic plasticity is enabled using spike-timing dependent plasticity (STDP), allowing the network to learn temporal structure in repeated stimulus sequences (S1 → S2 → S3). Neuromodulatory influences are incorporated through dopamine- and acetylcholine-dependent gain modulation, with acetylcholine enhancing bottom-up sensory processing during training. Following training, the network is evaluated under an omission paradigm in which expected stimuli are removed, enabling assessment of predictive coding dynamics. Simulations are executed using a differentiable JAX-based backend, allowing efficient parallelization across devices and seamless integration with gradient-based optimization.

## METHODS — ANALYSIS PIPELINE
Simulated neural activity is transformed into local field potential (LFP) signals and analyzed using a standardized multi-stage pipeline. Time–frequency representations are computed using spectral decomposition methods, enabling characterization of oscillatory dynamics across canonical frequency bands (theta, alpha, beta, gamma). Functional connectivity is assessed through coherence and correlation matrices, while directional interactions are quantified using Granger causality analysis to distinguish feedforward (gamma-mediated) from feedback (beta-mediated) communication. All analyses are aligned to event onset times (stimulus and omission), with baseline normalization performed over pre-stimulus intervals. The pipeline is applied identically to both simulated and empirical data, ensuring direct comparability and reproducibility. Output metrics include band-limited power, inter-area coherence, and hierarchical propagation patterns, which are used to validate the model against known electrophysiological signatures of predictive processing.
