# Sweep and Optimization Logs

Detailed execution logs for specific optimization tasks and parameter sweep experiments.

## Logs

- **baseline_opt_log.txt**: Log detailing the optimization process for the baseline E-I balance.
- **inter_area_sweep_log.txt**: Captures the progress of sweeps exploring interactions between brain areas.
- **ipnoise_sweep_log.txt**: Log for experiments analyzing the impact of IPnoise levels on activity.
- **laminar_100_opt_log.txt**: Detailed log of the parameter optimization for the 100-neuron laminar column.

These logs are essential for diagnosing optimization failures and confirming convergence.
